# types

Armazena todas as tipagens globais da aplicação.

Serve para padronizar estruturas de dados do sistema.

## O que colocar aqui

* tipos de entidades (Project, User, etc)
* tipos de API
* tipos de formulários
* enums
* interfaces compartilhadas

## Regras

* não colocar lógica
* não importar React
* apenas definição de tipos
* tipagens devem ser reutilizáveis

## Organização recomendada

Um arquivo por domínio:

project.ts
user.ts
sidebar.ts

# utils

Funções utilitárias puras e reutilizáveis.

Servem para operações genéricas que não dependem de React.

## O que colocar aqui

* formatação de dados
* cálculos
* helpers de string
* helpers de data
* transformações simples

## Regras

* funções devem ser puras
* não acessar API
* não usar React
* não depender de estado global

## Exemplos

formatCurrency()
formatDate()
parseNumber()
capitalize()
